Welcome to Ecommerce API

Ecommerce API is a project developed by Ismail Aslan. It provides a platform for users to register, login, manage products, and more.

Tech Stack

Ecommerce API is built using Express.js and PostgreSQL with Sequelize.js for the ORM. It also integrates Stripe for payment processing.

The project is deployed on a VPS and uses GitHub Actions for continuous integration and deployment.
